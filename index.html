Color Detection Project

I developed a Python project for real-time color detection system for individuals with Color Vision Deficiency (Color Blindness), using hand tracking and color correction techniques to accommodate abnormal lighting. OpenCV handled the video capture and color adjustments, while MediaPipe (a machine learning algorithm) detected hand gestures for user interaction, allowing the user to point at an object and be shown its color. The system calculates the correct color based on a scanned card (similar to White Cards in photography), and searches my generated SQL database for the most accurate color estimate.
